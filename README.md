### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nikhil2004nk/book-store-mng.git
cd book-store-mng
```

2. Install dependencies:

3. Start the development server:
